> 🌈 记录生活中的美好瞬间，分享学习与工作的点滴成长。

## 📖 项目简介

**开心快乐每一天** 是一个非经营性个人分享网站，记录生活小确幸、学习心得和工作经验。

## 🌐 在线访问

- **网站地址**: https://z979054461.github.io/happy-daily-website/

## ✨ 功能模块

| 模块 | 说明 |
|------|------|
| 🏠 首顴 | 网站主页，展示最新动态 |
| 🌈 生活 | 记录生活美好瞬间、日常小确幸 |
| 📚 学习 | 学习心得、知识点总结 |
| 💼 工作 | 工作经验分享、实用技巧 |
| 👤 关于我 | 个人介绍与联系方式 |

## 🛠️ 技术栈

- **前端**: HTML5 + CSS3 + JavaScript (无框架)
- **部署**: GitHub Pages + GitHub Actions
- **特性**: 响应式设计、深色模式、图片懒加载

## 📁 项目结构

```
happy-daily-website/
├── .github/workflows/     # GitHub Actions 自动部署
├── assets/               # 静态资源 (css/js/images/fonts)
├── life/                 # 生活分享模块
├── study/                # 学习分享模块
├── work/                 # 工作分享模块
├── index.html            # 首页
└── about.html            # 关于页面
```

## 🚀 本地运行

```bash
git clone https://github.com/z979054461/happy-daily-website.git
cd happy-daily-website
python3 -m http.server 8000
# 访问 http://localhost:8000
```
